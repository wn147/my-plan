공부계획

나는 우선 생활코딩에서 1.Web1 HTML&Internet 2.Web2 CSS 3.Web2 Java Script4.Web2 Node.js 5.Web2 Home server 6.Web2 Domain Name System
7.Web2 HTTP 8.Web2 OAuth 2.0을 완전히 배운뒤 이전까지 배운 것들을 활용하여 나만의 웹 어플리케이션 만들어 볼 예정이다.
이후 어느정도 웹에 익숙해지고 클론코딩 유튜브 채널을 참고하여 서비스 및 기획을 한뒤 앱을 만들어 볼 예정이다.이 과정에서 회사채용 공고등에서
요구로하는 능력을 주로 학습할 예정이다. 이 과정을 끝내면 코딩테스트를 대비하여 Leet Code에서 코딩을 연습한뒤 이력서등을 작성하여 준비를 할 예정이다.
